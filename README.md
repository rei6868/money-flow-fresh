# Money Flow - Personal Finance Management

## Setup

npm install
npm run dev

## Database

- Neon PostgreSQL (serverless)
- See `.env.local` for DATABASE_URL

## Project Structure

- `PROJECT-SPEC.md` - Database schema & API specs
- `AGENT-PROMPT.md` - Implementation guide
- `src/types/` - TypeScript types
- `src/lib/` - Database & utilities
- `src/app/` - Next.js pages & API routes
- `src/components/` - React components

## Development

Phase 1: Foundation (Types, DB, Validation, Calculations)
Phase 2: API Routes (CRUD operations)
Phase 3: UI Components (Frontend)
